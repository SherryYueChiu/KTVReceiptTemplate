# KTV 收據模板

一個風格化的 KTV 歌單記錄工具，模仿傳統油墨制單機的字體和風格，讓您與朋友去 KTV 後可以記錄當天的歌單，並生成懷舊風格的收據樣式表格。

## 📋 專案簡介

這個專案是一個純前端網頁應用，可以將 KTV 歌單轉換成復古收據樣式的表格。它使用了傳統收據機的等寬字體和紙張質感背景，營造出懷舊的視覺效果。

## ✨ 功能特點

- 🎵 **歌單記錄**：輕鬆記錄和編輯 KTV 歌單
- 🎨 **復古風格**：模仿傳統油墨制單機的字體和視覺風格
- 📝 **可編輯內容**：所有文字內容都可以直接編輯
- 🖼️ **紙張質感**：使用皺紋紙張紋理背景，增加真實感
- 📊 **自動統計**：自動計算歌曲數量和總時長
- 💾 **即時編輯**：點擊歌曲編號即可批量編輯歌單
- 🎯 **響應式設計**：適配不同螢幕尺寸

## 🚀 使用方法

1. **開啟專案**
   - 直接在瀏覽器中開啟 `index.html` 檔案

2. **編輯歌單**
   - 點擊歌曲列表中的編號（IDX）欄位
   - 在彈出的編輯框中輸入歌曲名稱（一行一首歌）
   - 點擊「儲存」按鈕更新歌單

3. **自訂內容**
   - 所有帶有 `contenteditable="true"` 屬性的文字都可以直接點擊編輯
   - 包括標題、KTV 名稱、日期、訂單號碼等

4. **查看結果**
   - 歌單會自動以收據樣式顯示
   - 包含歌曲編號、歌曲名稱、總數統計等資訊

## 🛠️ 技術棧

- **HTML5**：頁面結構
- **CSS3**：樣式設計和響應式布局
- **JavaScript (Vanilla)**：互動功能和歌單管理
- **自訂字體**：使用 `@font-face` 載入特殊字體

## 🎨 主要樣式特點

- **字體**：使用 `Merchant_Copy.ttf` 和 `Arial Narrow` 等字體模擬收據機效果
- **背景**：皺紋紙張紋理背景，增加真實感
- **顏色**：黑白配色，符合傳統收據風格
- **布局**：固定寬度的收據樣式容器

## 🔧 自訂說明

### 修改預設歌單

在 `index.html` 的 JavaScript 區塊中，找到 `songs` 陣列並修改：

```javascript
const songs = [
  "歌曲名稱1",
  "歌曲名稱2",
  // ... 更多歌曲
];
```

### 修改樣式

所有樣式都在 `index.html` 的 `<style>` 標籤中，您可以：
- 調整字體大小
- 修改背景圖片
- 改變顏色配置
- 調整容器寬度
